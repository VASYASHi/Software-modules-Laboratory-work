# Инструкция по проверке лабораторных работ

Лабораторные работы расположены в ветках:
  main — первая лабораторная,
  остальные ветки подписаны соответствующим образом.

Чтобы ознакомиться с кодом лабораторной работы, необходимо:

## Для первой лабораторной:

1. Перейти в папку src;
2. Открыть файл App.js;
3. Код лабораторной находится внутри функции App.
   Остальное можно игнорировать.

## Для второй и последующих лабораторных:

1. Перейти в нужную ветку (они пронумерованны в соотвествии!);
